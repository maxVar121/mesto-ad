# Проект Mesto

Финальная проектная работа курса. Интерактивное веб-приложение, в котором пользователь может редактировать профиль, обновлять аватар, добавлять и удалять карточки мест, ставить лайки и просматривать изображения в полноразмерном виде. Все данные синхронизируются с сервером через REST API.

## Опубликованный проект

🔗 [Открыть на GitHub Pages](https://USERNAME.github.io/REPO_NAME/)

> Не забудьте подставить свой логин и название репозитория.

## Технологии

- HTML5, CSS3 (БЭМ — Nested)
- JavaScript (ES6 modules, Fetch API)
- Vite — сборщик
- gh-pages — публикация

## Функциональность

- Загрузка данных пользователя и карточек с сервера
- Редактирование профиля (имя, описание)
- Обновление аватара
- Добавление и удаление собственных карточек
- Постановка и снятие лайков, отображение количества
- Просмотр карточки в полноразмерном модальном окне
- Подтверждение удаления карточки в отдельной модалке
- Валидация форм на чистом JS
- UX-улучшения: индикатор сохранения на кнопках, очистка ошибок при открытии форм

## Запуск проекта

Установка зависимостей:

```bash
npm install
```

Локальный сервер разработки:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

Публикация на GitHub Pages:

```bash
npm run deploy
```

## Настройка API

Перед запуском подставьте свой токен и идентификатор группы в файле `src/scripts/components/api.js`:

```js
const config = {
  baseUrl: "https://mesto.nomoreparties.co/v1/wff-cohort-NN",
  headers: {
    authorization: "ВАШ_ТОКЕН",
    "Content-Type": "application/json",
  },
};
```
